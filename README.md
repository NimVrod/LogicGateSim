# LogicGateSim

A visual logic gate simulator.

## Requirements

- CMake 4.1.1+
- C++23 compiler (MSVC 2022, GCC 13+, or Clang 17+)

## Build & Run

```bash
mkdir build
cd build
cmake ..
cmake --build . --config Release
```

Run the executable from the build directory:
```bash
# Windows
.\Release\LogicSimulator.exe
```
